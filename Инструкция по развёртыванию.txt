Программа написана на языке Python 3.8, для запуска требуется компилятор, который можно сказать по ссылке python.org 

Перед началом работы нужно сказать проект из репозитория https://github.com/Bull6/Test-NGP и запустить файл main.py. 

Для запуска программы нужно запустить командную строку (Win+R и ввести команду CMD для Windows, Сtrl+Alt+T для Linux(ubuntu)) 

Ввести в консоль python main.py (при необходимости изменить путь до файла)

Если нужно изменить входных данных, перейдите в папку input внутри проекта и измените файл input16.txt. Если требуется изменить путь к файлу или его имя, нужно открыть файл main.py в любом тестовом редакторе.
